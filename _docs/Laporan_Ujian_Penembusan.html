2. Perincian Pemulihan

2.1 Pembaikan Pintasan Pengesahan (Google Auth)

Isu Terdahulu: Fail google_auth.php mempercayai sebarang e-mel yang dihantar oleh klien tanpa pengesahan.
Tindakan Pembaikan: Kod telah dikemaskini untuk mengambil id_token daripada klien dan menghantarnya ke pelayan Google (oauth2.googleapis.com/tokeninfo) untuk pengesahan. Hanya token yang sah dan dijana untuk Client ID aplikasi ini sahaja diterima.

2.2 Pembersihan Kata Laluan Terdedah

Isu Terdahulu: Lebih daripada 25 fail PHP mengandungi baris kod kata laluan pangkalan data secara jelas.
Tindakan Pembaikan: Skrip automasi digunakan untuk membuang blok kod tersebut daripada kesemua fail terjejas dan menggantikannya dengan include 'config.php';.

2.3 Pengukuhan Muat Naik Fail

Isu Terdahulu: Fungsi muat naik fail hanya menyemak sambungan fail.
Tindakan Pembaikan: Fail vehicle_details.php kini menggunakan fungsi finfo_open(FILEINFO_MIME_TYPE) untuk memastikan fail adalah benar-benar imej sebelum disimpan.
