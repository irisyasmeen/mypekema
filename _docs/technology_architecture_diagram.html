Frontend

Built with semantic HTML5 and styled using utility-first TailwindCSS. Alpine.js provides lightweight reactivity for menus and modals.

Backend

Powered by PHP 8.2 running on Apache. Handles business logic, authentication sessions, and data processing.

Database

MySQL relational database stores vehicle inventories, user profiles, and analytical data with strict schema integrity.

Integrations

Leverages Google Cloud Platform for secure OAuth authentication and Maps visualization.
